Pixel Board Help

Tools

  • P - Pixel: Place single pixels
  • B - Brush: Paint with brush strokes
  • L - Line: Draw straight lines (click twice)
  • F - Fill: Flood fill regions
  • E - Eraser: Erase pixels
  • S - Spray: Paint dispersed spray
  • I - Picker: Sample colors
  • T - Text: Click canvas → type → Enter to commit

Controls

  • 1-9 - Select palette color (double-click to edit)
  • [/] - Change brush size
  • G - Toggle grid
  • Ctrl+Z / Ctrl+Shift+Z - Undo / Redo
  • +/- - Zoom in / out
  • 0 - Reset zoom
  • Arrow keys - Pan canvas
  • Shift+Click or Middle+Drag - Pan
  • Scroll - Zoom
  • Right-click - Pick color

About

This is a realtime collaborative pixel board. Any pixel you place can be overwritten by others. Last-write-wins. No permanent state. Enjoy the chaos.
